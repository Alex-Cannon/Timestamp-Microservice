Timestamp Microservice

Free Code Camp's APIs and Microservices Certificate project 1 of 5:
Convert time to and from unix or "natural" dates.
By Alex Cannon

See the Source Code on GitHub.


Example Usage:

  • GET /api/September 12, 2020

    Expected Response:
    {"unix":1599868800000,"natural":"September 12, 2020"}
  • GET /api/1599868800000

    Expected Response:
    {"unix":1599868800000,"natural":"September 12, 2020"}
