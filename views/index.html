Timestamp Microservice

Free Code Camp's APIs and Microservices Certificate project 1 of 5:
Convert time to and from unix or "natural" dates.
By Alex Cannon

See the Source Code on GitHub.


Example Usage:

  • GET /api/timestamp

    Expected Response:
    {"unix": "NOW_IN_UNIX","natural": "NOW_IN_UTC"}
  • GET /api/timestamp/2020-09-12

    Expected Response:
    { "unix": 1599868800000, "natural": "Wed Dec 09 2020 00:00:00 GMT+0000 (UTC)" }
  • GET /api/timestamp/1599868800000

    Expected Response:
    { "unix": 1599868800000, "natural": "Wed Dec 09 2020 00:00:00 GMT+0000 (UTC)" }
  • GET /api/timestamp/INVALID_DATE_STRING

    Expected Response:
    "Bad Request. Please use a valid date string."
